# Support UI
The Support UI is React library for my team

## Warning
🛠️🛠️🛠️This Library is in development🛠️🛠️🛠️

## Installation
using npm:
```js
$ npm install int2_support_ui
```

## Usage
```js
import SupportUI from "int2_support_ui";

const TestComponent = () => {
  return <SupportUI.SearchInput />
}
```
